## 0.5 ⚡ 最高原则:大规模并发(Agent Teams)

> **本质上这些工作绝大多数是可以并发的,必须派出【几十个 agent】用 Agent Teams 同时干,不要串行。**

- **7 条路线之间完全独立**——目录互不相干,天然可开 **7 个并行小队**,一队一路线,互不阻塞。
- **每条路线内部,R1–R13 大多也可并发**——不同功能落在不同文件/模块:
  渲染、PTY 接线、窗格 UI、命令面板+键位、标签页、配置/主题、CI 打包……各写各的文件即可并行。
  → 每条路线可再开 **多个队友**(典型 4–6 个),因此总规模轻松到 **几十个 agent 同时作战**。
- **用 Agent Teams,不用 workflow**:派**具名队友**(Agent 工具带 `name`),`teammateMode: in-process` 已开;
  主会话当 **team lead**,用 SendMessage 协调、收活、整合。
- **防冲突铁律**:每个队友**只写自己那份不重叠的文件**;**入口文件(main.rs/main.js/main.dart/main.cpp/main.go)与跨模块整合只由 lead 改**。需要入口接线的,队友在自己文件顶部用注释写清"lead 该如何调用我"。
- **并发粒度参考**(可据当下 token/机器并发上限伸缩):
  `7 路线 × (渲染 + PTY + 窗格 + 命令面板 + 标签页/配置 + CI 打包 + 验收) ≈ 40+ 个 agent`。
- **收敛靠 CI**:并发写完 → lead 整合 → 推 → 读 CI → 再派队友修,循环逼近达标(配合 ralph-loop)。

## 1. "生产级"的定义

一条路线达到生产级,意味着它是一个**真能用、能分发**的 Windows Terminal 式终端:能显示真 shell、能自由切分/交换窗格、有命令面板与标签页、能打包成可下载即跑的程序。逐条见 §3 的 R1–R13。

## 2. 验收纪律(所有 agent 必须遵守)

- **只有证据才能打 [x]。** 每勾一项,必须在该行末尾追加证据:CI 运行链接 / 截图路径 / 测试输出 / 命令输出。无证据的勾一律视为无效,复查时打回 `[ ]`。
- **CI 是编译/构建的唯一裁判。** 本机只有 `node`(无 cargo/go/dart/flutter/cmake),原生路线的"能编译"以 GitHub Actions 结果为准。
- **产物必须真能跑。** "打包完毕"= 有 `upload-artifact` 且下载解压后能启动(如 Electron 已验证)。仅 `cargo build` 过、无可运行产物 ≠ 达标。
- **网络约束:** `github.com` git 通道间歇性被封;推送/CI 触发只能在其可用时进行。被网络阻塞的项标注 `⛔ 阻塞(网络)` 并跳到下一个可动的项,不要空转。
- **不破坏已冻结的基线:** tag `electron-baseline-v0.1`(commit `7ac6e79`)与 `_builds/electron-win-v0.1/` 是已交付的干净 Electron 版,**不得改动/覆盖**。
- **提交规范:** commit 信息与 PR **绝不出现 "claude"/Anthropic 字样,不加 Co-Authored-By**;**绝不写任何开源协议/License**;不改 `ReadMe.md` 的大小写。
- **每完成一项就提交一次**(小步提交),commit 信息引用本文件的条目号(如 `R2/Src_Tauri`)。

## 5. Agent 工作协议(ralph-loop 每轮照做)

1. **读**本文件,找**最靠前的未完成 `[ ]`**(优先 §4 基础设施,再 §3 各路线;Electron 已达标项跳过)。
2. **做**这一项:改代码/改 CI。文件分工不重叠时可派具名队友(Agent Teams)并行,主会话当 lead 整合。
3. **验证**:能本地验的本地验(node 自测/浏览器可视化/截图);要编译的推 CI(网络可用时),读 CI job 结论/日志。
4. **打勾**:验证通过才把 `[ ]` 改 `[x]` 并在行末**附证据**;失败则留 `[ ]` 并追加一行 `↳ 尝试N:<失败原因>`。
5. **提交**:小步 commit(信息引用条目号,无 claude/无 License)。
6. **循环**:回到第 1 步。遇网络阻塞的项标 `⛔` 跳过,永不空转。
7. **收尾**:当本文件所有 `[ ]` 均为 `[x]` 时,输出 `WIN_TERM_MAC_ALL_PRODUCTION_DONE`。
